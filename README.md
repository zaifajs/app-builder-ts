# Fidelidade App Builder

A test project from 2019 (developed at Comon) - an interactive platform that promotes consistency across mobile applications in the Fidelidade insurance ecosystem. This wizard helps users discover and configure the right type of mobile app based on their needs through a series of guided questions.

## About

The Fidelidade App Builder is a decision-making tool that guides users through questions to determine which app type best fits their requirements. It supports various app types including:

- **Product Apps** - Apps for managing products and customer relationships (e.g., My Savings)
- **Service Apps** - Apps for service management
- **ProAssistance Apps** - Professional assistance applications
- **Beyond Apps** - Extended functionality apps

The platform provides detailed information about UX guidelines, pattern libraries, component specifications, and technical requirements for each app type, ensuring consistency across the Fidelidade mobile app ecosystem.
